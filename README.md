# Русификатор MMO StickyFirstPost
Русский язык плагина MMO StickyFirstPost для XenForo 2

## Требования плагина
* XenForo 2.1

## Актуальная версия
  * XenForo 2.1.11
  * MMO StickyFirstPost 2.0.2

## Описание плагина
Позволяет закреплять первое сообщение в теме.
